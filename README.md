# ChenYuLab-Website

This repository contains the **frontend code** of the Chen Lab official website.  
It is built using **Vue 3 + Vite** and provides a clean, maintainable structure for a research lab website.

---

## 📌 Key Features

- **Home Page** — Lab introduction  
- **Research Page** — Wet-lab & dry-lab research  
- **Team Page** — Members with profiles and lab photos
- **Publications** — Year-based publications
- **Contact Page**

---


## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
